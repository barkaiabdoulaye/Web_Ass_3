# User Stories - SMEStack

## SME Owner Stories

1. **As a** SME owner, **I want to** post a project request **so that** I can find students to help me.
   - *Acceptance:* Form with title, description, skills, budget

2. **As a** SME owner, **I want to** see student applications **so that** I can choose the best candidate.
   - *Acceptance:* List of applicants with their messages

3. **As a** SME owner, **I want to** approve completed work **so that** I can ensure quality.
   - *Acceptance:* Button to approve each milestone

## Student Stories

4. **As a** student, **I want to** browse open projects **so that** I can find work.
   - *Acceptance:* List of projects with search and filter

5. **As a** student, **I want to** apply to projects **so that** I can get selected.
   - *Acceptance:* Application form with message

6. **As a** student, **I want to** update my progress **so that** the SME knows what I've done.
   - *Acceptance:* Progress update form

## Admin Stories

7. **As an** admin, **I want to** assign students to projects **so that** work can begin.
   - *Acceptance:* Assignment interface

8. **As an** admin, **I want to** see all activity **so that** I can monitor the platform.
   - *Acceptance:* Activity log with timestamps
