# Street Report - SMEStack (Musanze Skill-Share)

## User Quote (from Musanze context)

> "I need a website for my shop, but professional agencies are too expensive. INES students have the skills, but how can I find them and be sure the work will be done well?"
> — Marie Uwase, Owner of "Musanze Crafts" (Local SME)

## The Exact Problem Being Solved

Small businesses in Musanze need digital services (websites, apps) but cannot afford professional agencies. INES students need real projects to build their portfolio. Currently, there is no platform to connect them.

## Chosen Scenario

**SMEStack - Musanze Skill-Share**

A platform that connects local SMEs with INES students for digital projects.

## Scope for This Assignment

- User login with 3 roles (SME, Student, Admin)
- SMEs can post project requests (CRUD)
- Students can browse and apply to projects
- Admin can assign teams
- Milestone tracking (5 stages)
- Progress bar visualization
- Role-specific dashboards
- Activity logging
