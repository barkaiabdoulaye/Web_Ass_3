# Project Scope - SMEStack

## In Scope ✅

### Features
- User registration and login (3 roles)
- SMEs: Post, edit, delete project requests
- Students: Browse, search, apply to projects
- Admin: Assign teams, manage users
- Milestone tracking (5 stages)
- Progress bar with JavaScript
- Dashboards for each role
- Activity logging

### Technical
- MVC architecture
- MySQL with prepared statements
- Responsive design (mobile + desktop)
- 25+ Git commits
- 12+ test cases

## Out of Scope ❌

- Online payments (will be added later)
- Real-time chat
- Mobile app
- Multi-language support
